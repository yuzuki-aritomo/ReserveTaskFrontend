# Reserve Next App

## 初回 Setup

```bash
$ git clone git@github.com:yuzuki-aritomo/ReserveTaskFrontend.git

$ cd ReserveTaskFrontend

$ docker compose build

$ docker compose up -d
```

## docker起動

```bash
$ docker compose up -d
```

├── next-app #プロジェクト
